# 1，获得对方好感的话术

**首因效应**：第一印象很重要。三大因素：外貌，声音，语言。开场：你好，很高兴认识你。

**握手效应**：先握手再开始交谈。

**微笑效应**：爱笑的人运气总是很好

**相似性效应**：找到相似的地方，拉近彼此的距离

**近因效应**：从指责开始，以称赞结尾。例子：虽然不善言辞，但是给人的印象很有教养。

**基本归因错误**：当对方犯错，尝试从外部因素中寻找问题。

**情感融通**：倾听对方的话，模仿对方的行为，适当分享自己的私人故事。



# 2，获得对方好感的话术

**认知失调理论**：帮助过你一次的人，以后会更愿意帮助你。

**道歉理论**：勇敢地承认自己的错误并请求原谅。道歉六要素：表示后悔，解释错在哪里，**承认错误**，保证不再发生，提出补偿或者补救措施，请求原谅。

**感觉适应效应**：不要昨天吵架，今天就立刻主动讨好，时间是必要的，应该循序渐进和解。

**上帝情结**：学会换位思考，不要一言堂。

**相互性法则**：先打招呼，有去的情才能有来的情。

**后见之明偏差**：知道结果认为一开始就知道了哪件事情的结果倾向。永远不要说“我就知道会这样”。

**过度自信效应**：评价自身能力高于平均水平的倾向。对吹牛的人睁一只眼闭一只眼。

**邻里效应**：不要和别人比较才能恢复信心



# 3，让对方说YES的话术

**权威法则**：引用名人的话，让自己的话更有分量

**兰格实验**：为了说服别人，给出理由。例子：我希望提高我的年薪，因为我工作非常努力，有资格得到更好的待遇。

**安慰剂效应**：用坚定的自信感武装自己，让对方产生期待心理。

**蔡格尼克效应**：越重要的话越不要一次性说完

**框架效应**：先展现出积极的框架。

**面包香味效应**：烤面包的香味能诱导陌生人做出更积极、亲切和利他的心理现象。

**斯托克代尔悖论**：直视并承认悲观的现实，但同时相信克服这种现实可以在未来取得好的结果。



# 4，让异性缘变好的话术

**犯错误效应**：比起完美的人，人们更喜欢有一点缺点的人。

**罗密欧与朱丽叶效应**：男女之间的爱情具有在遇到障碍或周围的反对时，反而更加牢靠的特性。例子：没事，虽然你闺蜜反对我们在一起，但是就算重来我也会坚定地选择你一千次一万次。

**社会认同理论**：采用多数人的选择作为支持自己想法和行动的根据。例子：很多阿姨都夸我是靓仔的 。

**对比效应**：虽然价值相同，但是可以根据比较而感觉到不同差异的错觉。例子：给我你们这里最贵的蜜雪冰城饮料。

**舌尖现象**：因为紧张，无法将储存在脑海中的信息说出口。例子：我很紧张是因为喜欢你

**智性恋**：不留痕迹地表现自己不凡的博学和教养。多读书！



# 5，让顾客打开钱包的话术

**象征效应：** 购买特定产品，就认为自己会成为使用该产品群体中的一员的心理。例子：这个是周杰伦在使用的同款产品。

**锚定效应**：先抛出高定价，再用低折扣吸引顾客的话术。

**狄德罗效应**：人们在购买了一件物品后，想要继续购买与之搭配的物品。

**乐队花车效应**：从众效应。跟随流行而进行消费商品。

**热手效应**：通过夸奖顾客幸运使其欣然消费的话术。

**古德曼定律**：倾听不满并努力解决从而留住顾客。

**心理感应抗拒理论**：用限购方法催促顾客赶紧购买的话术。



# 6，激励团队提高效率的话术

**赋予进步效应**：鼓励学生继续努力的话术。开始就是成功的一半。

**穆拉文实验**：激发员工热情和自律的话术。意志力不是维持不变的，而是会随着使用而消耗。

**公开表明效应**：将自己的目标告诉周围的人，增加自己的执行力，从而更好地实现目标。

**问题行为效应**：通过自问自答的方式来提高实践能力

**霍桑效应**：表明自己正在关注，以激励对方提升效率的话术

**林格曼效应**：集体成员人数越多，人均贡献度就越低。解决办法：经常直呼其名。

**罗伯斯山洞实验**：即使是不同性质的群体，只要有共同的目标，矛盾就会减少。
